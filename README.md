# repo-viewer

## 简介

repo-viewer是一款开源免费的源代码仓库浏览软件,与同类型软件相比,支持多平台的桌面系统，如UOS ARM等，提供SVN、GIT等常用源代码仓库的浏览操作，比如文件、提交日志、属性查看等。[ Repo-viewer is an open-source and free code repository viewer. Compared with similar software, it supports a variety of desktop systems, including UOS ARM. It provides browsing operations for commonly used code repositories such as SVN and GIT, including file viewing, commit log viewing, and property viewing. ]

软件基于electron开发，遵循MIT开源协议。

## 安装说明

推荐直接使用绿色版程序压缩包，解压即用。当使用安装程序进行安装时会被360安全卫士误告警 “有程序试图修改应用关键程序及DLL”，告警的提示中可见修改的都是本软件自己的库及文件，忽略即可。

npm run dist
